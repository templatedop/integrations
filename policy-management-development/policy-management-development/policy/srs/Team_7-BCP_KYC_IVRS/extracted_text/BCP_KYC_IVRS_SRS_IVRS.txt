1. Overview

Interactive voice response (IVR) is a technology that allows a computer to interact with humans through the use of voice and DTMF tones input via keypad.
In Insurance Industry IVR systems can respond with prerecorded or dynamically generated audio to further direct users on how to proceed. IVR applications can be used to control almost any function where the interface can be broken down into a series of simple interactions. 
This document provides an overview of the Services, which will be invoked once the customer is authenticated, IVR presents all the self service options to the customer, from which customer can choose a specific request for PLI System to respond to.
 


2. Scope and Assumptions

1. IVR will interact with PLI Solution via Biztalk using Web Service Route. 
2. IVR System will be integrated with PLI solution's APIs using XML over HTTP mechanism.
3. The authentication of customer requesting a specific Service will be handled by IVR System. PLI system will share the date of birth of customer in response to policy number given as request. The validation of date of Birth will be done by IVR system.
4. In case of the technical/business error, IVR system is expected to send the same request again[HI1].[AG2]
